# Network Monitor

A simple network monitoring tool to track network connectivity, latency, and performance metrics.

## Features

- Monitor network connectivity
- Track latency and response times
- Log network statistics
- Alert on connection issues

## Requirements

- Python 3.7+
- See `requirements.txt` for dependencies

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python network_monitor.py
```

## Configuration

Edit the `config.json` file to customize monitoring targets and settings.
